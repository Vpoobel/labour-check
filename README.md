# Labor Check · Presentación web

Presentación web ejecutiva para Comité de Dirección.

## Uso

Abre `index.html` en un navegador moderno.

También puedes servirla localmente:

```bash
cd labor_check_presentacion_web
python -m http.server 8000
```

Después abre `http://localhost:8000`.

## Controles

- Flecha derecha / espacio: siguiente sección
- Flecha izquierda: sección anterior
- P: imprimir o exportar a PDF
